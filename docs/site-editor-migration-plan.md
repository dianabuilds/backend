- Интерфейс доступен в административном разделе по маршруту /management/site-editor. Таблица использует преднастроенный шаблон Table и отображает заголовок, slug, тип, статус, локаль, владельца и версии.
- Фильтры: поиск по названию/slug, тип, статус, наличие черновика, локаль и сортировка (updated_at_desc|updated_at_asc|title_asc). Значения передаются напрямую в GET /v1/site/pages.
- Карточка детали (data-testid="site-page-detail") синхронизирована с выделением строки и показывает метаданные (версии, владельца, дату обновления, статус ревью).
- Покрыто e2e-сценарием cypress/e2e/site_editor_catalog.cy.ts, который проверяет загрузку списка, работу фильтра по статусу и перестроение карточки.
