# PageHero

PageHero — геройский блок страницы административного интерфейса. Он объединяет заголовок, действия, фильтры и ключевые KPI, чтобы пользователь сразу видел контекст и мог перейти к важным действиям.

## Варианты

- **default** — универсальный вариант для большинства разделов. Сочетает светлый фон и мягкие градиенты.
- **metrics** — акцент на визуализации метрик. Используется на дашбордах и в разделах Observability.
- **compact** — компактный блок для вложенных страниц и карточек настроек.

## Рекомендации

- Настраивайте maxHeight, если внутри героя появляются дополнительные панели (например, развернутый фильтр).
- В мобильной вёрстке оставляйте только один CTA; остальные перемещайте в нижнюю панель действий.
- Для KPI используйте сокращённые числа (через ormatNumber) и понятные подписи.
